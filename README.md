# Customer Revenue & Retention Analysis

## Project Overview
This project analyzes customer revenue distribution and retention behavior using transactional data.

## Tools Used
- Excel (Data understanding)
- SQL (Data transformation & analysis)
- Power BI (Dashboard & visualization)

## Key Insights
- Revenue is evenly split between repeat and non-repeat customers (~50/50)
- Monthly revenue trends are stable across segments
- Average rating is 3.0 → indicates improvement opportunity
